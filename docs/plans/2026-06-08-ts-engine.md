## 关键注意（与 Python 版的差异）

- 事件接口取名 **`GameEvent`**，避开 DOM 全局 `Event`。
- 字段 **camelCase**：`revealPhase` / `revealedInfo` / `publicEvents` / `privateEvents`。
- `revealedInfo` 用 `Set<string>`。
- VisibilityGate 铁律：`private` 对 owner 恒可见；`directed` 须 owner **且** `revealedInfo.has(id)`；`public` 须 revealed；`omniscient` 永不返回。
- 测试断言**不变量**（无他人秘密 / 无真相 / 无未来阶段线索），不比对文本。
